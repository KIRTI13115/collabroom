\chapter*{ABSTRACT}

The increasing demand for remote communication and collaboration has led to the widespread adoption of video conferencing platforms. This project presents the design and development of a video conferencing web application built using modern web technologies such as Next.js and TypeScript.

The system provides secure user authentication using Clerk, allowing users to sign in through social login or email-based authentication. Once authenticated, users can create new meetings, join existing sessions, and access a wide range of real-time communication features including screen sharing, recording, participant management, and interactive controls.

The application integrates real-time communication services using Stream, enabling seamless audio-video interaction among participants. It also includes features such as scheduling future meetings, maintaining a history of past meetings, and providing a responsive user interface that adapts across devices.

The system is designed with a focus on scalability, security, and performance. By leveraging modern frontend frameworks and real-time APIs, the application ensures low latency and high reliability.

This project demonstrates the practical implementation of real-time web technologies in building scalable communication systems and highlights the potential for further enhancements such as AI-based meeting summaries and advanced analytics.
